# Blood Pressure Tracker

A responsive, user-friendly web application designed to help users log, monitor, and track their daily blood pressure readings over time.

---

## 🚀 Features

* **Easy Data Entry:** Quick inputs for Date, Systolic, and Diastolic pressure.
* **Real-Time Assessment:** Instantly calculates and displays your "Current Reading Level" status based on health standards.
* **Reading History:** Automatically logs entries into a clean, structured table for tracking trends.
* **Responsive Design:** Fully styled container layout that works beautifully across different screen sizes.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for the tracking form and history table.
* **CSS3:** Clean, modern interface with a soothing color palette and centered layout.
* **JavaScript (ES6):** Handles data input, status categorization logic, and dynamic table updates.

---

## 📂 File Structure

* `blood_pressure.html` - The main user interface and structure.
* `blood_pressure.css` - The styling and layout definitions.
* `blood_pressure.js` - The application logic and interactivity.

---

## 🧑‍💻 How to Use

1. Clone or download this repository.
2. Open the `blood_pressure.html` file in any modern web browser.
3. Enter the reading date, systolic, and diastolic numbers, then click **Add Reading**.
